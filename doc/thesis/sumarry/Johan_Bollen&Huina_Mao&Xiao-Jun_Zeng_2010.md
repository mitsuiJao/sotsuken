# Twitterの気分指標が株式市場を予測する

**原題**: Twitter mood predicts the stock market
**著者**: Johan Bollen, Huina Mao, Xiao-Jun Zeng
**掲載誌・会議 / 年**: Journal of Computational Science, Vol. 2, Issue 1, 2011 / arXiv 2010
**arXiv / DOI**: arXiv:1010.3003v1

> Twitter投稿から抽出した「冷静さ（Calm）」次元の気分指標がDJIA株価変動を3〜4日先行して予測し、予測精度87.6%を達成。

---

## 背景・問題設定

効率的市場仮説（EMH）は株価がランダムウォークに従い予測不能と主張するが、EMHの前提に疑義を呈する先行研究が蓄積されている。一方、行動経済学・行動ファイナンスは感情・気分が個人および集合的な意思決定に重大な影響を及ぼすことを示す。感情がミクロな投資行動を通じてマクロな株価動向にも波及する可能性がある。

従来の公衆気分測定手段（Gallup世論調査・消費者信頼感指数）はコスト・タイムラグが大きく、リアルタイム株価予測には不向きである。Twitterは大規模かつ低コスト・リアルタイムな公衆気分のサロゲートとして注目されつつあったが、多次元的気分指標による株価予測への応用は未開拓だった。

本研究は、Twitterフィードから抽出した公衆気分時系列がDJIA（ダウ工業平均株価）の変動と相関・予測的関係にあるかを検証することを目的とする。

## 提案手法

### データ
- 収集期間：2008年2月28日〜12月19日（約985万ツイート、約270万ユーザー）
- 気分表明を示す自己言及的フレーズ（"I feel", "I am feeling" 等）を含むツイートを抽出
- DJIA日次終値：Yahoo! Financeより取得

### 気分計測ツール

**OpinionFinder（OF）**：感情極性（ポジティブ/ネガティブ）を日次時系列として出力。2,718語の肯定語・4,912語の否定語レキシコンを使用。

**GPOMS（Google-Profile of Mood States）**：心理測定尺度POMS-biを基に、Googleの5-gram統計（約1兆語）で語彙を64語→964語に拡張した独自ツール。以下の6次元の日次時系列を生成：
$$\text{Calm, Alert, Sure, Vital, Kind, Happy}$$

全時系列はスライディングウィンドウ $k$ 日のz-スコアに正規化：
$$Z_{X_t} = \frac{X_t - \bar{x}(X_{t\pm k})}{\sigma(X_{t\pm k})}$$

### 分析フレームワーク（3フェーズ）

1. **クロスバリデーション**：2008年10月〜12月の気分時系列を大統領選挙・感謝祭の既知イベントで検証
2. **グレンジャー因果性分析**：以下の2モデルを比較してDJIA予測能を評価
   - $L1: D_t = \alpha + \sum_{i=1}^n \beta_i D_{t-i} + \varepsilon_t$（ベースライン）
   - $L2: D_t = \alpha + \sum_{i=1}^n \beta_i D_{t-i} + \sum_{i=1}^n \gamma_i X_{t-i} + \varepsilon_t$（気分変数追加）
3. **自己組織化ファジーニューラルネットワーク（SOFNN）**：非線形モデルで予測性能を定量評価

## 実験・評価

**期間設定**：
- Granger因果性分析：2008年2月28日〜11月3日（大統領選・感謝祭を除外、DJIA時系列64日）
- SOFNNの訓練：2月28日〜11月28日、テスト：12月1日〜12月19日（安定期・異常イベントなし）

**SOFNNの入力組み合わせ**：

| 入力 | 内容 |
|------|------|
| $I_0$ | DJIA過去3日のみ（ベースライン） |
| $I_{\text{OF}}$ | DJIA + OpinionFinder |
| $I_1$ | DJIA + Calm（$X_1$） |
| $I_{1,2}$ | DJIA + Calm + Alert |
| $I_{1,3}$ | DJIA + Calm + Sure |
| $I_{1,4}$ | DJIA + Calm + Vital |
| $I_{1,5}$ | DJIA + Calm + Kind |
| $I_{1,6}$ | DJIA + Calm + Happy |

**評価指標**：MAPE（平均絶対パーセント誤差）と方向正解率（上昇/下落）

## 結果

### クロスバリデーション
- OFは大統領選（11/4）・感謝祭（11/27）のポジティブ感情急上昇を正確に検出
- GPOMSは選挙前の不安増大（Calm低下）、選挙日の正常化（Vital・Happy・Kind急上昇）、感謝祭のHappyスパイクと多次元的反応を詳細に捉えた

### グレンジャー因果性分析

| ラグ | OF | Calm | Alert | Sure | Vital | Kind | Happy |
|------|-----|------|-------|------|-------|------|-------|
| 2日 | 0.268 | **0.013** | 0.973 | 0.811 | 0.369 | 0.991 | 0.706 |
| 3日 | 0.436 | **0.022** | 0.981 | 0.349 | 0.418 | 0.991 | 0.723 |
| 4日 | 0.218 | **0.030** | 0.998 | 0.415 | 0.475 | 0.989 | 0.750 |
| 5日 | 0.300 | **0.036** | 0.989 | 0.544 | 0.553 | 0.996 | 0.173 |

Calm（$X_1$）のみがラグ2〜6日でDJIAに対して統計的に有意なグレンジャー因果性（$p < 0.05$）を示した。OFを含む他の6次元は有意でなかった。

### SOFNN予測結果

| 入力 | MAPE (%) | 方向正解率 (%) |
|------|----------|--------------|
| $I_{\text{OF}}$ | 1.95 | 73.3 |
| $I_0$（ベースライン） | 1.94 | 73.3 |
| $I_1$（+Calm） | 1.83 | **86.7** |
| $I_{1,2}$（+Calm+Alert） | 2.03 | 60.0 |
| $I_{1,3}$（+Calm+Sure） | 2.13 | 46.7 |
| $I_{1,4}$（+Calm+Vital） | 2.05 | 60.0 |
| $I_{1,5}$（+Calm+Kind） | 1.85 | 73.3 |
| **$I_{1,6}$（+Calm+Happy）** | **1.79** | 80.0 |

Calm単独（$I_1$）が最高の方向正解率86.7%を達成。Calm+Happy（$I_{1,6}$）はMAPEを最低（1.79%）に抑え方向正解率も80%と高水準。OFの追加はベースラインに対して改善をもたらさない。

なお、87.6%正解率（20日間で15日以上正解）の偶然一致確率は3.4%であり、チャンスレベルではないと統計的に確認。また、Calm+HappyをCalmのみと比較したF検定（$p=0.66$）では線形モデルで優位差が出ないことから、両者の相互作用が非線形であることが示唆される。

## 結論・今後の課題

Twitter投稿の大規模集合から抽出した公衆気分指標のうち、GPOMS「Calm」次元が3〜4日のラグでDJIAの変動に対してグレンジャー因果的であることを示した。単純なポジティブ/ネガティブ二値分類（OpinionFinder）では捉えられない多次元的気分構造が株価予測に有効である。SOFNNへの統合により方向正解率87.6%・MAPEを6%以上改善することを示した。

**限界**：
- 地理的限定（2008年のTwitterユーザーは英語圏・米国中心）
- 公衆気分の「グラウンドトゥルース」が不明
- 株価への因果メカニズムは未解明（気分→投資行動→株価という経路の検証は今後の課題）

**将来展望**：
- 位置情報・多言語対応による地域サンプリングバイアスの解消
- より高度な予測モデル（ニュース・経済指標との統合）への応用
- Twitterを公衆幸福度調査の補完ツールとして活用

## 一言コメント

感情の多次元性（Calm vs. Happy等）が株価予測に非線形的に寄与するという発見は、単純なセンチメント二値分類の限界を鋭く指摘しており示唆に富む。2010年の論文ながら、Twitterセンチメントと株価予測分野の礎となった先駆的研究であり、現代のLLMベース感情分析研究への理論的出発点として参照価値が高い。
