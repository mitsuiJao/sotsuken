# LLMセンチメント分析による S&P 500 トレーディング戦略の強化

**原題**: Enhancing Trading Performance Through Sentiment Analysis with Large Language Models: Evidence from the S&P 500
**著者**: Haojie Liu, Zihan Lin, Randall R. Rojas（UCLA）
**掲載誌・会議 / 年**: arXiv preprint, 2025
**arXiv / DOI**: arXiv:2507.09739v1 [q-fin.CP] (2025年7月13日)

> GPT-2・FinBERT のセンチメントスコアとMACD等のテクニカル指標・時系列モデルを融合したハイブリッド戦略が、S&P 500 のバイ＆ホールドを大幅に上回る 5.77% のリターンを達成した。

---

## 背景・問題設定

株式市場では、テクニカル指標や時系列モデル（ARIMA・ETS・Prophet）が予測に広く利用されてきたが、これらはリアルタイムのニュースや市場センチメントを捉えるのが苦手という本質的な限界を持つ。効率的市場仮説（EMH）はすべての情報が株価に即座に織り込まれると主張するが、実証的には市場反応が不完全なケースが多数報告されている。

NLP技術の発展により、FinBERT・GPT-2 といった金融特化型・汎用大規模言語モデル（LLM）を用いたニュースセンチメント分析が実用化され、Twitter/X や金融ニュースのセンチメントが株価変動と有意な相関を持つことが複数の先行研究で示された（Gu & Kurov 2020; Kirtac & Germano 2024; Greyling & Rossouw 2025）。

本研究は、複数の金融ニュースソースから取得したセンチメントスコアをMACDなどのテクニカル指標・時系列予測モデルと統合し、S&P 500 の取引戦略を最適化することを目的とする。ニュースセンチメントと従来手法を組み合わせることで、予測精度と取引パフォーマンスが向上するかを包括的に検証する。

## 提案手法

### データ

- **株価データ**: S&P 500（ˆGSPC）の日次価格（Yahoo Finance、2019年8月8日〜2024年8月7日）
- **ニュースデータ**: Wall Street Journal（WSJ）、Barron、Benzinga、MarketWatch、Dow Jones の記事（日付・タイトル・本文）
- **ラグ構造**: 16:00〜23:59 ET 配信記事は翌営業日に紐付け、週末記事は月曜リターンに集約。ラグ $k=1$（当日ニュース→翌日リターン）が最良。

### センチメント指標

**GPT-2**: ドロップアウト層（rate=0.5）を追加したファインチューニング版。ポジティブ/ニュートラル/ネガティブの3クラスで日次センチメントを生成。

**FinBERT**: 金融コーパス事前学習済みモデル（dropout rate=0.8）。金融特有の語彙・リスク表現の理解に優れる。

複数記事が同日に存在する場合は**多数決（mode）**で日次センチメントを決定。

### テクニカル指標

$$\text{MACD Line} = EMA_{12} - EMA_{26}, \quad \text{Signal Line} = EMA_9(\text{MACD Line})$$

| 指標 | 概要 |
|------|------|
| MACD | 12/26日EMA差分 + 9日シグナル線のクロスで売買判断 |
| SAR | $\text{SAR}_t = \text{SAR}_{t-1} + \alpha(\text{EP} - \text{SAR}_{t-1})$、トレンド反転を検知 |
| VW MACD | 出来高加重MACDで流動性の高い局面でのシグナル信頼性を向上 |
| Dual MACD | 短期（12-26-9日）と長期（19-39-9日）の二重確認でダマシを低減 |

### 時系列モデル

- **ARIMA**: $y'_t = \phi_1 y'_{t-1} + \cdots + \theta_q \epsilon_{t-q} + \epsilon_t$
- **Prophet**: $y(t) = g(t) + s(t) + h(t) + \epsilon_t$（トレンド・季節性・イベント効果を分解）
- **ETS**: $y_t = (T_t \cdot S_t) \cdot \epsilon_t$（誤差・トレンド・季節性の指数平滑）

### トレーディングシミュレーション

初期資本 $C_0 = \$10{,}000$。センチメントとテクニカル指標を統合した合成シグナル：

$$I_t = \frac{\sum I_{it}}{|\sum I_{it}|}$$

$I_t > 0$ で買い、$I_t < 0$ で売り、$I_t = 0$ でホールド。テスト期間：2024年5月10日〜8月7日。取引コスト・スリッページはゼロを仮定。

## 実験・評価

- **評価指標**: センチメント分類精度（accuracy）、時系列予測精度（accuracy）、累積リターン（%）
- **ベースライン**: バイ＆ホールド戦略（テスト期間の実績リターン = −0.696%）
- **実装**: センチメント分析は Python（Transformers・PyTorch）、取引戦略・時系列は R（quantmod・TTR・forecast・prophet）

## 結果

### センチメント分類精度

| ニュースソース | FinBERT (%) | GPT-2 (%) |
|---------------|-------------|-----------|
| Benzinga      | **75.56**   | 64.44     |
| Dow Jones     | **67.69**   | 60.00     |
| WSJ           | 57.14       | **65.48** |
| Barron        | 50.63       | **64.56** |
| MarketWatch   | 32.22       | 34.44     |

FinBERT は Benzinga・Dow Jones で優位、GPT-2 は Barron・WSJ で優位。

### 時系列・テクニカル指標精度

| 指標 | 精度 (%) |
|------|----------|
| Prophet | **59.65** |
| ETS     | **59.65** |
| ARIMA   | 29.82    |
| MACD    | 7.02     |
| SAR     | 7.02     |
| VW MACD | 5.26     |
| Dual MACD | 3.51  |

テクニカル指標単体の予測精度は10%未満と低い。

### 取引リターン（主要結果）

| 戦略 | リターン (%) |
|------|-------------|
| GPT Dow Jones + VW MACD | **5.77** |
| GPT Dow Jones + Dual MACD | 5.38 |
| GPT Dow Jones + MACD | 5.10 |
| GPT-2 + VW MACD ARIMA (Dow Jones) | 4.39 |
| FinBERT Benzinga + Dual MACD | 4.64 |
| バイ＆ホールド（ベンチマーク） | −0.696 |

センチメント単独では安定しないが、テクニカル指標（特にVW MACD・Dual MACD）との組み合わせで顕著な改善が得られた。GPT-2 は Dow Jones データとの組み合わせで最高リターンを達成し、FinBERT は Benzinga での精度面では優位。

## 結論・今後の課題

GPT-2・FinBERT によるリアルタイムセンチメント分析をMACDやSAR等のテクニカル指標・ARIMA/ETS時系列モデルと統合したハイブリッド戦略は、バイ＆ホールドを大幅に凌駕する 5.77% のリターンを実現した。FinBERT の金融語彙理解力が高精度センチメントを可能にし、取引精度の向上に寄与した。

**限界**:
- 取引コスト・スリッページをゼロ仮定しており実運用での収益を過大評価
- 記事の重み付けが均等（長さ・著者信頼性・ソースバイアス不考慮）
- バックテスト期間が短い（2024年5〜8月のみ、強気相場や暴落局面未検証）
- S&P 500 指数のみ対象、個別株や他資産クラスへの汎化未検証

**将来展望**: 強化学習による動的戦略最適化、X/Redditのリアルタイムソーシャルメディアデータ追加、LSTM・Transformerの深層学習モデルへの拡張、他資産クラスへの一般化。

## 一言コメント

センチメント単体・テクニカル指標単体では低精度（≤10%）であることを明確に示しつつ、両者の組み合わせで相乗効果が生じる点が実証的に興味深い。ただし、バックテスト期間わずか3ヶ月かつ取引コストゼロ仮定は結論の信頼性を大きく制限しており、実運用への適用には慎重な検討が必要。GRU/Transformerとの比較や、FinBERTとGPT-2で最適ニュースソースが異なる理由の分析が今後の課題として残る。
