# ルールベースの株取引推薦システム：センチメント分析とテクニカル指標の統合

**原題**: A Rule-Based Stock Trading Recommendation System Using Sentiment Analysis and Technical Indicators
**著者**: Yuri Kim, Sujin Yoo, Seongbin Park（高麗大学・漢城大学）
**掲載誌・会議 / 年**: Electronics (MDPI), 2025, Vol. 14, No. 773
**DOI**: https://doi.org/10.3390/electronics14040773

> ニュース感情スコア（TextBlob）とRSIを組み合わせたルールベースの売買推薦システムが、主要市場指数を上回る5〜7%台のリターンと89%超の正解率を個人投資家向け軽量設計で達成した。

---

## 背景・問題設定

株価は多数の要因に左右される複雑なシステムであり、従来の時系列モデルは高ボラティリティ環境で有効性が低い。近年、NLPを用いたセンチメント分析が定量金融戦略に取り込まれているが、既存システムには以下の課題があった。

1. **単独利用の限界**: センチメント分析のみ（テクニカル指標なし）またはテクニカル分析のみのアプローチが多く、補完的な視点を欠く
2. **高コスト**: ハイエンドGPUや高額トレーディングプラットフォームを要求し、個人投資家に不向き
3. **小規模データ**: 先行研究（[19]）は8記事しか使わず代表性に問題

本研究は「リアルタイムのニュースセンチメント分析とRSIを統合した、個人投資家向け軽量・低コストの売買推薦システムを構築できるか」という問いに答える。開発・テストは第12世代 Intel Core i7-1260P（2.10 GHz）・16 GB RAMの一般ノートPCで実施された。

## 提案手法

### 入力と全体フロー

ティッカーシンボル（例：`AAPL`）を入力として受け取り、以下の2系統のデータを並行取得・分析する（Figure 1）。

- **株価データ**: Polygon.io から過去2ヶ月の日次OHLCV（直近30取引日に注目）
- **ニュースデータ**: NewsAPI から直近7日分の記事（最大100件）のタイトル・説明文

### テクニカル分析（RSI）

$$RSI = 100 - \frac{100}{1 + RS}, \quad RS = \frac{\text{Average Gain over } n \text{ periods}}{\text{Average Loss over } n \text{ periods}}$$

$n=14$（デフォルト）。RSI > 70 で買われ過ぎ、RSI < 30 で売られ過ぎの判定。

### センチメント分析（TextBlob）

TextBlobでニュース記事の極性（polarity）を計算。7日間の平均を1〜100にスケーリング：

$$\text{Scaled Sentiment} = (\text{Average Sentiment} + 1) \times 50$$

50が中立、高値は強気、低値は弱気を示す。

### 売買判断ルール

| 推薦       | 条件                                         |
|-----------|----------------------------------------------|
| Strong Buy  | センチメント > 70 かつ RSI < 30               |
| Buy         | センチメント > 50 かつ 30 ≤ RSI < 50          |
| Neutral     | 40 ≤ センチメント ≤ 60 かつ 40 ≤ RSI ≤ 60   |
| Sell        | センチメント < 50 かつ RSI > 70               |
| Strong Sell | センチメント < 30 かつ RSI > 70               |

理論的根拠はダウ理論（相場6局面）とグランビル投資則（局面別の売買タイミング）。

### トレーディングアルゴリズム

- BUY/Strong BUYシグナルで翌日始値に購入（追加購入は前回購入価格を下回った場合のみ）
- NeutralまたはSell系シグナルへ転換時に売却（利益率 ≥ 1%、または14日以上保有後は0.25%で可）

## 実験・評価

**初期バックテスト**: 2024年9月11日〜10月31日（7週間）、米国株95銘柄（大型34・中型30・小型31）、Polygon.io + NewsAPI使用。

**拡張実験**:
- 期間: 2週間・1ヶ月（終了日: 2025年1月24日）
- 時価総額別: 大型・中型・小型各10銘柄
- ベータ別: 高ベータ（β≥1）・中ベータ（0<β<1）・負ベータ（β≤0）各10銘柄

評価指標: 平均RoR・中央値・標準偏差・CV・95%信頼区間・歪度・尖度・正解率（収益方向の一致率）。ベンチマークはNASDAQ（COMP）・DJIA・S&P 500。

## 結論・今後の課題

センチメント分析とRSIを組み合わせたルールベースシステムが、軽量設計にもかかわらず複数の時間枠・株カテゴリにわたって主要市場指数を上回るリターンを達成。89%超の正解率も実証された。

**Limitations**:
- 短期戦略に特化しており、長期指標（業績・マクロ経済）は未対応
- 取引手数料・スリッページを考慮していない（Sharpe比等のリスク調整指標も未計算）
- 外部イベント（地政学的リスク・政策変更）への対応が不十分

**Future Work**:
- 基本的財務指標・マクロ経済データの統合
- ストップロス・ポートフォリオ分散によるリスク管理機能
- ニュースソース信頼性のウェイト付け・脱バイアスアルゴリズムの導入

## 一言コメント

TextBlobというシンプルなルールベースNLPとRSIの組み合わせでも、適切な売買ロジックを設計すれば個人投資家レベルのハードウェアで市場平均超過リターンが得られることを実証した点が興味深い。ただし取引コスト・スリッページ非考慮・限定バックテスト期間という制約は大きく、実運用への適用には慎重な検証が必要。センチメントにLLMを活用したり、他の感情辞書（VADER等）と比較したりする発展も自然な次ステップとして考えられる。
