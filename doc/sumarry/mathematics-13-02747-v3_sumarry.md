# FinBERT感情分析・SHAP解釈可能性・差分プライバシーを統合した株価予測

**原題**: Stock Price Prediction Using FinBERT-Enhanced Sentiment with SHAP Explainability and Differential Privacy
**著者**: Linyan Ruan, Haiwei Jiang
**掲載誌・会議 / 年**: Mathematics, 2025, Vol. 13, 2747
**DOI**: https://doi.org/10.3390/math13172747

> FinBERTで抽出した金融ニュース感情をXGBoostに統合しSHAPで解釈することで、技術指標単独より12.6%高いAUCと2倍超の模擬PnLを達成した。

---

## 背景・問題設定

株価予測は非定常・高次元・ノイズの多い金融時系列の性質から困難であり、ARIMAやGARCHといった伝統的計量経済モデルは非線形依存性の捕捉が苦手である。XGBoostなどの勾配ブースティング決定木（GBDT）は優れた汎化性能を示すが、数値シグナルのみに依存し、ニュース・アナリストコメントなどの非構造化情報を活用できない。

金融テキストの感情分析は、ドメイン特有の専門用語（例："depreciation"は一般語では否定的だが金融では中立的）が原因で、VADER・TextBlobなどの汎用ツールでは誤分類が頻発する。FinBERT（Financial PhraseBank上でファインチューニングされたBERT）はこの問題を解消し、金融テキストの感情分類においてSOTAを実現する。

XGBoostなどのブラックボックスモデルは金融規制・受託責任上の透明性要件を満たさない。加えて、クライアント取引データや専有ニュースフィードを使う実運用では差分プライバシー（DP）によるメンバーシップ推論攻撃への防護が求められる。本研究はこれら3つの課題（予測精度・解釈可能性・プライバシー）を一体的に解決するマルチモーダルフレームワークを提案する。

## 提案手法

### 感情特徴量の構築

FinBERTは各ヘッドライン $h$ に対し確率ベクトル $\text{FinBERT}(h) = (P_\text{pos}, P_\text{neu}, P_\text{neg}) \in \Delta^2$ を出力する。スカラー感情スコアを以下で定義する：

$$s(h) = \alpha P_\text{pos}(h) - \beta P_\text{neg}(h)$$

実装では $\alpha = \beta = 1$、指数減衰重み $\lambda = 1.5$（検証セットでのAUCチューニング）を採用。各取引日 $t$ について加重平均・加重分散・範囲・超過尖度からなる4次元感情ベクトル $s_{s_i}^{(t)} \in \mathbb{R}^4$ を構築する。

### 構造化特徴量

対数リターン $r(t)$、移動平均 $\text{MA}(k)$、実現ボラティリティ $\hat{\sigma}_k(t)$、RSI、MACD、出来高変化などの技術指標を構築。感情特徴量と連結し全マルチモーダル入力 $x^{(t)} \in \mathbb{R}^d$ とする。

### 学習目標

翌日終値が上昇するかどうかの二値分類：

$$y(t) = \mathbf{1}\left[P_\text{close}(t+1) > P_\text{close}(t)\right]$$

XGBoostの正則化付きクロスエントロピー損失を最適化：

$$L(\theta) = \sum_{(x,y)\in D} \ell(y, f_\theta(x)) + \sum_{m=1}^M \Omega(f_m), \quad \Omega(f_m) = \gamma T_m + \frac{1}{2}\lambda \sum_{j=1}^{T_m} w_j^2$$

### 差分プライバシー（DP-SGD）

$(ε, δ)$-差分プライバシーを以下で定式化：

$$\forall S, \forall D \sim D' : \Pr[\mathcal{M}(D) \in S] \leq e^\varepsilon \cdot \Pr[\mathcal{M}(D') \in S] + \delta$$

ミニバッチ $B$ で per-sample 勾配をクリッピングノルム $C$ で切り詰め、ガウスノイズを付加：

$$\tilde{g}_i = \text{clip}(\nabla_\theta \ell(x_i, y_i), C), \quad \bar{g} = \frac{1}{|B|}\sum_{i\in B}\tilde{g}_i + \mathcal{N}(0, \sigma^2 C^2 I)$$

Moments Accountant法で総合的なプライバシー消費量を追跡。

### SHAP解釈可能性

TreeSHAPを用いてShapley値を効率的に厳密計算：

$$f_\theta(x^{(t)}) = \varphi_0 + \sum_{j=1}^d \varphi_j^{(t)}, \quad \varphi_j^{(t)} = \sum_{S \subseteq F\setminus\{j\}} \frac{|S|!(d-|S|-1)!}{d!}\left[f_\theta(x_{S\cup\{j\}}) - f_\theta(x_S)\right]$$

効率性・対称性・ゼロ性・線形性の4公理を満たし、グローバル重要度は $I_j := \frac{1}{N}\sum_{t=1}^N |\varphi_j^{(t)}|$ で定義。

## 実験・評価

- **データ**: S&P 500の主要銘柄（AAPL・MSFT・TSLA・JPM）、2018年1月〜2023年12月
- **株価データ**: Yahoo Finance（yfinance）、OHLCV日次データ
- **ニュースデータ**: Reuters・Bloomberg・MarketWatchなど（FNSPID公開データセット含む）；市場終値後のヘッドラインは翌日に割り当て（情報漏洩防止）
- **評価**: ローリングウィンドウ（学習24ヶ月・テスト6ヶ月、5折）
- **ベースライン**: T（技術指標のみ）、T+M（技術+モメンタム）、T+V（VADER感情）
- **モデル設定**: 最大木深さ6、学習率0.05、ブースティング300ラウンド、サブサンプル比0.8
- **市場レジーム**: Pre-COVID（2018-01〜2020-01）、COVID暴落（2020-02〜04）、Post-COVID回復（2020-05〜2021-11）、インフレ・利上げ（2021-12〜2023-12）

## 結果

### モデル性能比較（5ローリングウィンドウ平均）

| モデル | Accuracy | F1 | AUC | Precision | PnL (%) |
|--------|----------|-----|-----|-----------|---------|
| T（技術指標のみ） | 0.624 | 0.608 | 0.652 | 0.601 | 3.21 |
| T+M（+モメンタム） | 0.636 | 0.620 | 0.661 | 0.615 | 3.87 |
| T+V（VADER感情） | 0.644 | 0.631 | 0.669 | 0.622 | 4.85 |
| **T+F（FinBERT感情）** | **0.703** | **0.688** | **0.740** | **0.678** | **7.63** |
| T+F−V（ボラティリティなし） | 0.689 | 0.674 | 0.722 | 0.666 | 6.93 |
| T+F−M（モメンタムなし） | 0.676 | 0.663 | 0.715 | 0.656 | 6.57 |

T+FはT比でAUC+12.6%、PnL+138%。T+VとT+FのAUC・F1差は99%信頼水準（$p < 0.01$）で統計的有意。

### SHAP特徴量グループ貢献度

| 特徴量グループ | SHAP重要度 (%) |
|----------------|---------------|
| FinBERT感情（平均・最大・標準偏差） | **28.6** |
| ボラティリティ指標（GARCH・ローリング標準偏差） | 21.4 |
| モメンタム指標（MA・RSI・MACD） | 17.3 |
| 価格リターン | 13.8 |
| 出来高特徴量 | 9.2 |

### 市場レジーム別AUC / 模擬PnL

| モデル | COVID暴落期 | 強気相場期 | 停滞期 |
|--------|-------------|-----------|--------|
| T | 0.608 / 1.2% | 0.642 / 3.6% | 0.593 / 2.4% |
| T+V | 0.635 / 2.0% | 0.661 / 4.2% | 0.614 / 2.7% |
| **T+F** | **0.729 / 4.9%** | **0.748 / 7.8%** | **0.723 / 6.2%** |

停滞期にT・T+VがAUC 0.59〜0.61に低下する一方、T+Fは0.723を維持。

### 差分プライバシーと性能トレードオフ（$\delta = 10^{-5}$）

$\varepsilon \geq 0.5$ でベースラインAUCの95%以上を保持（AUC=0.699）。$\varepsilon = 1.0$ でAUC=0.717まで回復。

### 汎化性能（銘柄別 平均AUC / AUC標準偏差）

| 銘柄 | T | T+V | T+F |
|------|---|-----|-----|
| AAPL | 0.624 / SD 0.031 | 0.645 / 0.027 | **0.713 / 0.018** |
| MSFT | 0.618 / 0.035 | 0.641 / 0.030 | **0.701 / 0.021** |
| JPM | 0.591 / 0.038 | 0.610 / 0.034 | **0.674 / 0.025** |
| TSLA | 0.648 / 0.040 | 0.667 / 0.037 | **0.735 / 0.020** |

T+Fはレジーム間でのAUC標準偏差が最小（平均0.021）、すなわち最も安定。

## 結論・今後の課題

FinBERT感情特徴量はXGBoostの予測性能を全指標で大幅改善し、SHAP分析で最重要特徴群（28.6%）と確認された。差分プライバシー機構は実用的なプライバシー水準（$\varepsilon \geq 0.5$）で性能劣化を最小限に抑制する。本フレームワークは複数銘柄・複数市場レジームで安定した性能を示し、実運用への展開適性が高い。

**Limitation**: 大型株中心のため中小型株への汎化は未検証；ニュースソースの信頼性重み付けは未実装；PnL計算は取引コスト・スリッページを含まない理想化条件。

**今後の課題**: マルチ資産ポートフォリオ・高頻度パイプライン・決算コール等リッチなテキストソースへの拡張；情報源信頼性を考慮した重み付け感情集約；感情非対称性パラメータ（$\alpha, \beta$）の市場レジーム適応的学習。

## 一言コメント

FinBERT・XGBoost・TreeSHAP・DP-SGDを単一フレームワークに統合した実用性重視の構成で、金融NLPにおける解釈可能性と差分プライバシーの共存を実証した点が評価できる。ただし模擬PnLが取引コストを無視している点には注意が必要で、実運用での収益性は過大評価の可能性がある。本論文はFinBERT感情特徴量がテクニカル指標よりSHAP重要度で上位を占める知見を提示しており、感情×ボラティリティのインタラクション効果はリスク管理観点でも示唆に富む。
