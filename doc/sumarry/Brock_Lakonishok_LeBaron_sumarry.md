# 単純テクニカル売買ルールと株式リターンの確率的性質

**原題**: Simple Technical Trading Rules and the Stochastic Properties of Stock Returns
**著者**: William Brock, Josef Lakonishok, Blake LeBaron
**掲載誌・年**: The Journal of Finance, Vol. XLVII, No. 5, December 1992
**DOI**: —（JSTOR 収録論文）

> 移動平均・レンジブレイクアウトの売買シグナルはランダムウォーク等の帰無モデルでは説明できない超過リターンを1897〜1986年DJIAデータ上で実証した。

---

## 背景・問題設定

テクニカル分析は学術界から長らく否定されてきたが、1980年代後半以降の株式リターン予測可能性に関する研究群（Fama-French, De Bondt-Thaler 等）が効率的市場仮説に対する再考を促した。本論文はその流れを受け、最も広く使われる2種のテクニカルルール——**移動平均オシレーター**と**トレーディングレンジブレイクアウト**——が将来の価格変化を予測できるかを厳密に検証する。

先行研究の多くは短い期間・限られたルールセットでの検証にとどまり、データスヌーピングバイアスの問題が指摘されてきた。本論文はこれに対し、(1) 全ルールの結果を報告、(2) ダウ工業平均90年間（1897〜1986）という超長期データを利用、(3) 複数の非重複サブ期間で結果の頑健性を確認、という3点で対処する。

また、従来のt検定が正規・定常・時間独立分布を仮定するのに対し、本論文は**ブートストラップ手法**を組み合わせることで、帰無モデルから生成したシミュレーション分布と実データの統計量を比較する。これにより、複数ルールにまたがる統合検定と、非正規・ボラティリティクラスタリング等の現実的性質への対処を同時に実現している。

## 提案手法

### データ
ダウ工業平均（DJIA）の日次終値、1897年1月1日〜1986年12月31日（25,036観測値）。リターンは対数差分 $r_t = \log(P_t) - \log(P_{t-1})$ として計算。4つの非重複サブ期間（1897-1914, 1915-1938, 1939-1962, 1962-1986）でも検証。

### テクニカルルール

**1. 可変長移動平均（VMA: Variable-Length Moving Average）**

短期移動平均が長期移動平均を上抜けた日を買いシグナル、下抜けた日を売りシグナルとして分類し、その日の保有期間中のリターンを集計する。バンド幅（1%）を設定することでホイップソー回避の変形版も検証。パラメータセット：(短期, 長期, バンド) = (1,50,0), (1,50,0.01), (1,150,0), (1,150,0.01), (5,150,0), (5,150,0.01), (1,200,0), (1,200,0.01), (2,200,0), (2,200,0.01) の計10ルール。

**2. 固定長移動平均（FMA: Fixed-Length Moving Average）**

移動平均クロスオーバー発生後10日間のリターンを記録する方式。10日間中に発生した新規シグナルは無視。

**3. トレーディングレンジブレイクアウト（TRB: Trading Range Break）**

価格が直近50・150・200日の高値（抵抗線）を上抜けたとき買い、安値（支持線）を下抜けたとき売りシグナルとし、その後10日間のリターンを評価。バンド有無で計6ルール。

### ブートストラップ手法

4つの帰無モデルのもとでシミュレーション系列を各500本生成し、実データの統計量がシミュレーション分布のどの分位に位置するかでp値を推定する：

1. **ランダムウォーク（ドリフト付き）**: 実リターンをスクランブル（独立同一分布として再抽出）
2. **AR(1)**: $r_t = b + \rho r_{t-1} + \varepsilon_t$（OLS推定残差を再抽出）
3. **GARCH-M**: $r_t = a + \gamma h_t + b\varepsilon_{t-1} + \varepsilon_t$、$h_t = \alpha_0 + \alpha_1 \varepsilon_{t-1}^2 + \beta h_{t-1}$
4. **Exponential GARCH（EGARCH）**: $h_t = \alpha_0 + g(z_{t-1}) + \beta h_{t-1}$、$g(z_t) = \theta z_t + \omega(|z_t| - (2/\pi)^{1/2})$

統計量は買いリターン、売りリターン、買い-売り差、および各条件下の標準偏差の計5指標。

## 実験・評価

**基本統計量（Table I）**: 日次リターンは強い尖度（全期間Kurtosis=16.00）と弱い負の歪度を示す。自己相関は全体として小さいが、一部サブ期間（特に1939-1962）では有意。

**t検定（Section II）**: 買い平均リターン・売り平均リターンそれぞれを無条件平均と比較するt統計量、および買い-売り差のt統計量を報告。

$$t_{buy-sell} = \frac{\mu_b - \mu_s}{(\sigma^2/N_b + \sigma^2/N_s)^{1/2}}$$

**ブートストラップ検定（Section IV）**: 各ルール・各帰無モデルについて、「シミュレーション500本のうち何本が実データの統計量を上回ったか（Fraction > Dow）」をp値として報告。

## 結果

### 基本検定結果（VMAルール、全期間）

| 統計量 | 値 |
|---|---|
| 買いシグナル日の平均日次リターン | +0.042%（年率約+12%） |
| 売りシグナル日の平均日次リターン | −0.025%（年率約−7%） |
| 無条件平均日次リターン | +0.017% |
| 買い期間の日次標準偏差 | 0.89% |
| 売り期間の日次標準偏差 | 1.34% |

全10ルールで買い-売り差はすべて正、対応するt値は高度に有意（買い-売り差の平均t統計量は4〜6台）。売りリターンはすべて負。FMAでは10日間の買い-売り平均差0.77〜1.09%（バンド有無）でTRBとともに同様の傾向。

### ブートストラップ結果の要点

- **ランダムウォーク**: 500本中買いリターンを上回るシミュレーションは0〜1本（p値≈0.000〜0.010）。買い-売り差のp値もほぼ0。→ **完全棄却**
- **AR(1)**: 買い-売りスプレッドはある程度生成されるが、Dowの0.067%に対してAR(1)は0.019%（p値=0.00）。→ **棄却**
- **GARCH-M**: 買いリターンのp値は0.086〜0.116程度とやや大きくなるが、売りリターンのp値は0.986〜1.000。実データの「売り期間の負リターン」を再現できず。→ **棄却**
- **EGARCH**: 売りリターンのp値は0.998〜1.000（全ルールでほぼ1.00）。買い-売り差p値≈0.00。→ **棄却**

いずれのモデルも買い-売りボラティリティ差（買い期間は低ボラ、売り期間は高ボラ）を再現できず、特にGARCH-Mは買い期間ボラを著しく過大推定（1.14% vs 実際0.89%）。

## 結論・今後の課題

90年間のDJIAを用いた検証の結果、移動平均・トレーディングレンジブレイクアウトの両テクニカルルールは株式リターンの予測可能性を示す強力なエビデンスを提供した。この予測可能性は以下のいずれによっても説明されない：

- ランダムウォーク（ドリフト付き）
- AR(1)（正の自己相関が買い-売り差の10%未満しか説明しない）
- GARCH-M（条件付き平均・分散ともに実データと乖離）
- EGARCH（売り期間の負リターンを再現不能）

特に売りシグナル後の**持続的負リターン**は、既存の均衡モデルでは解釈が困難であると著者らは指摘する。一方、実際の取引コストを考慮すれば戦略の実現可能性は限定的であるとも述べており、将来の研究課題として「なぜテクニカルルールが機能するのか」のメカニズム解明を挙げている。

## 一言コメント

1992年時点でブートストラップと複数帰無モデルを組み合わせた統計設計は方法論的に先進的で、この論文が「テクニカル分析を真面目に扱う嚆矢」となった背景がよくわかる。ただし現代の視点では、取引コスト・市場インパクト・サバイバーシップバイアスへの対処が薄い点、および90年間のデータ全体を使いながら帰無モデルパラメータを同一データから推定する点（内生性）が議論の余地として残る。Sullivan, Timmermann & White (1999) による"Data Snooping, Technical Trading Rule Performance, and the Bootstrap"と合わせて読むと議論の深みが増す。
